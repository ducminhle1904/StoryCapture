# Summary

Implemented the first hybrid Editor / Post-Production slice:

- Editor now has UI and Code modes, defaulting to UI.
- UI mode projects parsed `.story` scenes/commands into editable blocks and writes canonical DSL back to the same source buffer.
- Added `<story>.polish.json` load/save helpers and inline polish controls for recipe, zoom, callout, highlight, scene transition, cursor, and auto-zoom.
- UI mode sidecar writes are gated until the user changes polish settings; opening a project does not create a default sidecar.
- Parse IPC now exposes per-command `step_id` so polish data can be keyed to stable script steps.
- `Record & Polish` routes through the recorder and opens Post-Production after completion.
- Post-Production loads polish sidecar data, auto-generates callout/zoom clips, and defaults to Review & Export mode with Fine tune available.
- v1 polish timing is estimated across story steps and centered from the nearest trajectory frame when available; raw recordings remain clean.
