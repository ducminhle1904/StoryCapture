Structured-output field on request:

Provider struct `OpenAiProvider { http: reqwest::Client, api_key: Redacted<String> }`. Same client-builder pattern as Plan 04 (rustls, pool limits, 120s timeout).

Request body:
```rust
#[derive(Serialize)]
struct OpenAiRequest<'a> {
    model: &'a str,
    stream: bool,
    messages: Vec<serde_json::Value>,     // passthrough from LlmRequest
    max_tokens: u32,
    temperature: f32,
    #[serde(skip_serializing_if = "Option::is_none")]
    tools: Option<Vec<serde_json::Value>>,
    #[serde(skip_serializing_if = "Option::is_none")]
    tool_choice: Option<serde_json::Value>,
    #[serde(skip_serializing_if = "Option::is_none")]
    response_format: Option<serde_json::Value>,
    stream_options: StreamOptions,
}

#[derive(Serialize)]
struct StreamOptions { include_usage: bool }
```

Headers: `Authorization: Bearer <key>`, `Content-Type: application/json`.

SSE loop: `resp.bytes_stream().eventsource()`. Each `Event { data, .. }`:
- If `data == "[DONE]"` → break.
- `serde_json::from_str::<Value>(&data)` → drill into `choices[0].delta`.
- If `delta.content: String` → `LlmEvent::TextDelta`.
- If `delta.tool_calls: Vec<ToolCallDelta>`, accumulate `arguments` per `index` in `HashMap<u32, (String /*name*/, String /*args_buf*/)>`. On chunk where `choices[0].finish_reason == "tool_calls"`, flush accumulators to `LlmEvent::ToolUseComplete`.
- If top-level `usage` field (final chunk): emit `LlmEvent::Usage`.

On 429: read `retry-after`, return `LlmError::RateLimited`. On 401: `AuthFailed`.

Add `pub mod openai;` to `llm/mod.rs`.
