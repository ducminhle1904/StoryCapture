```rust
pub enum SelectorIssue {
    TooGeneric,              // .btn, #id, single-token class
    MissingFallback,         // no alt_selectors in args and selector is class/id
    DeepNthChild,            // :nth-child(N) depth >= 3
    AbsoluteXPath,           // /html/body/div[1]/...
    OverlyDynamicClass,      // contains hashed token like _a3f9b2
    BrittleAttribute,        // [style="..."] or [data-*] with generated value patterns
}

pub struct SelectorWarning {
    pub issue: SelectorIssue,
    pub message: String,
    pub suggestion: Option<String>,
}

pub fn analyze_selector(selector: &str, has_fallback: bool) -> Vec<SelectorWarning>;
```

Fixture format (YAML):
```yaml
- selector: ".btn"
  has_fallback: false
  expected: [TooGeneric, MissingFallback]
- selector: "button[data-testid='signup']"
  has_fallback: false
  expected: []
- selector: "/html/body/div[2]/span"
  has_fallback: false
  expected: [AbsoluteXPath]
```

Evaluation: for each fixture, `analyze_selector` returns a set of issues; compare against `expected`. TP = issue in both, FP = in output but not expected, FN = in expected but not output. Compute precision = TP / (TP + FP), recall = TP / (TP + FN). MUST meet thresholds.

```rust
#[test]
fn e11_confusion_matrix_meets_thresholds() {
    let broken = load_yaml("known_broken.yaml");
    let good = load_yaml("known_good.yaml");
    let mut tp = 0u32; let mut fp = 0u32; let mut fnn = 0u32;
    for entry in broken.iter().chain(good.iter()) {
        let found: HashSet<_> = analyze_selector(&entry.selector, entry.has_fallback).into_iter().map(|w| w.issue).collect();
        let expected: HashSet<_> = entry.expected.iter().cloned().collect();
        tp += found.intersection(&expected).count() as u32;
        fp += found.difference(&expected).count() as u32;
        fnn += expected.difference(&found).count() as u32;
    }
    let precision = tp as f64 / (tp + fp).max(1) as f64;
    let recall    = tp as f64 / (tp + fnn).max(1) as f64;
    assert!(precision >= 0.80, "precision {} < 0.80", precision);
    assert!(recall    >= 0.70, "recall {} < 0.70",    recall);
}
```
