## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-20-01 | Information Disclosure | Key input visible in DOM | mitigate | `type="password"` on input; onPaste handler clears clipboard after 30s via `navigator.clipboard.writeText("")` optional enhancement; `autocomplete="off"` |
| T-03-20-02 | Information Disclosure | DevTools inspection of key value in React state | mitigate | Do NOT hold raw key in React state post-save; `setApiKey` function immediately calls `key_set` then clears local state; test asserts state reverts to `""` after save |
