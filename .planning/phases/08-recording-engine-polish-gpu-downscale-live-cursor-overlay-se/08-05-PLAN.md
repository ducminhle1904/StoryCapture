<interfaces>
Shader contract (both platforms): cursor skin texture at slot `t1` (BGRA, RGBA-source pre-swizzled on upload — or swap in shader; pick ONE strategy and document), sampler `s0`, input src `t0`, output UAV `u0`, cbuffer `b0`:
```
{
  uint2  src_size;
  uint2  dst_size;
  float2 cursor_pos_dst;   // output-pixel coords; negative x means "no cursor"
  uint2  cursor_size;      // skin dims in pixels
}
```

Per-pixel logic (identical on both platforms; matches `compose_frame_into_bgra`):
```
uv = (float2(dtid.xy) + 0.5) / float2(dst_size)
base_bgra = src.SampleLevel(samp, uv, 0)         // bilinear
rel = int2(dtid.xy) - int2(cursor_pos_dst)
if (cursor_pos_dst.x >= 0 &&
    rel.x >= 0 && rel.x < int(cursor_size.x) &&
    rel.y >= 0 && rel.y < int(cursor_size.y)) {
  skin_bgra = skin_tex.Load(int3(rel, 0))         // no bilinear on skin
  if (skin_bgra.a > 0) {
    // straight-over: out_rgb = skin_rgb*skin_a + base_rgb*(1-skin_a)
    out = lerp(base_bgra, float4(skin_bgra.rgb, 1.0), skin_bgra.a)
  } else {
    out = base_bgra
  }
} else {
  out = base_bgra
}
dst[dtid.xy] = out
```

**Swizzle decision (locked for this plan):** On upload, the `skin.pixels` source is RGBA. Re-pack the bytes to BGRA BEFORE upload (pure-Rust one-pass: `for px in chunks_exact_mut(4) { px.swap(0, 2); }` on a clone, since `compose_frame_into_bgra` already swaps in the CPU path). Shader then reads `skin_tex.Load` → already-BGRA. This keeps the shader simple and matches the CPU pixel math.

Cursor sampler from 08-04: `sample_os_cursor() -> (f32, f32)` already exists in `pipeline.rs`. 08-05 adds: construct `gpu_scale::CursorOverlay { skin_bgra: &uploaded_bgra_bytes, skin_width, skin_height, pos_x, pos_y }` and pass as `Some(...)` to the scaler on the GPU-path arms.

Skin upload caching: the scaler's GPU-side skin texture is created lazily on the first `Some(CursorOverlay)` call per recording, keyed on `(skin_width, skin_height, bytes.as_ptr())`. Subsequent frames reuse it. When the pointer changes (shouldn't happen in-recording — skin is immutable), re-upload.

       struct Uniforms {

           texture2d<float, access::sample> src [[texture(0)]],
