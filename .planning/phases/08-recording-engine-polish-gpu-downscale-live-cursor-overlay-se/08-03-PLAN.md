Resolves Assumption A3 (WGC shared-texture feasibility) with a spike task at the start.

**Purpose:** Windows parity with 08-02 — the `-vf scale` CPU pass goes to D3D11 compute; WGC surface hands a D3D11 texture straight into the scaler without a CPU copy.
**Output:** Working `D3D11Scaler` + WGC native-surface path + pump branch for `NativeWindows` + 3 ms/frame bench + roundtrip tests + A3 spike result documented.
