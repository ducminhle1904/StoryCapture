<objective>

  <behavior>
    - `BrowserDriver` trait is `#[async_trait] Send + Sync`, with methods: `launch(config: LaunchConfig) -> Result<()>`, `close()`, `new_context()`, `new_page()`, `goto(url: &str)`, `click(selector: ResolvedSelector)`, `type_text(selector: ResolvedSelector, text: &str)`, `scroll(direction: ScrollDir, amount: Option<f32>)`, `hover(selector: ResolvedSelector)`, `drag(from: ResolvedSelector, to: ResolvedSelector)`, `select_option(selector: ResolvedSelector, value: &str)`, `upload_file(selector: ResolvedSelector, path: &Path)`, `wait_ms(ms: u64)`, `wait_for(target: WaitForTarget, timeout_ms: u64)`, `assert_present(target: AssertTarget)`, `screenshot(name: &str, out_dir: &Path) -> PathBuf`, `capabilities() -> CapabilitySet`, `current_cursor_position() -> Result<(i32, i32)>`, `resolve_selector(sel: &SelectorOrText) -> Result<ResolvedSelector>` (returns a driver-specific handle — the trait uses a `ResolvedSelector` enum that each driver interprets internally).
    - `LaunchConfig`: `{ url: Option<String>, viewport: Viewport, theme: Theme, headless: bool, base_url: Option<String>, download_dir: PathBuf }`
    - `CapabilitySet`: flags: `can_upload: bool`, `can_wait_for_download: bool`, `can_pierce_shadow_dom: bool`, `can_handle_oauth_popup: bool`, `supports_iframes: bool`, `supports_network_idle: bool`. `ChromiumoxideDriver::capabilities()` returns `{ upload: false, wait_for_download: false, pierce_shadow_dom: false, oauth_popup: false, iframes: true, network_idle: false }`. `PlaywrightSidecarDriver::capabilities()` returns all-true.
    - `Executor::run(story, primary_driver, fallback_driver)` inspects each command's required capability via `capability::required_for(&command)`; if primary lacks it, dispatches to fallback. Never fails silently — logs dispatch choice via `tracing::info!`.
    - Auto-wait wrapper (D-12): every action = `resolve → wait_actionable → act`. `wait_actionable(selector, timeout)` polls element's: visibility (CSS `visibility: visible`, no `display:none`, opacity > 0), stability (bounding box unchanged for 2 consecutive 100ms ticks), viewport (scrolled into view), not-animating (no CSS transitions running).
    - Smart selector (D-13): for `SelectorOrText::Text("Save")`, tries in order: (1) `text=Save` visible-text search via CDP `DOM.performSearch`, (2) `[data-testid="Save"]`, (3) `[aria-label="Save"]`, (4) CSS literal `Save` (will fail but logged). For `SelectorOrText::Selector(css)`, tries CSS only. For `Testid(id)`, tries testid only. For `Aria(label)`, tries aria only. Each attempt logged: `AttemptLog { strategy, value, outcome: Found | NotFound | Error(msg), elapsed_ms }`.
    - Meta-driven launch (AUTO-04): `meta.app` → `base_url` for relative `navigate` paths; `meta.viewport` → window size; `meta.theme` → emulated `prefers-color-scheme`.
    - `executor.rs` produces `mpsc::Receiver<ExecutorEvent>` and drives story end-to-end, persisting via injected `ProjectDb` writer (use `tokio::sync::Mutex<ProjectDb>` or actor channel — tests use a null writer).
    - Test pages are served from a tiny `tokio`-based static HTTP server spawned by tests on an ephemeral port (no external services). Test fixtures cover: button click, text input, file upload, shadow DOM element, scroll-into-view, drag+drop.
    - Chromium binary: tests use a system `chromium` / `chrome` binary located via `chromiumoxide::browser::Browser::launch`. If unavailable in CI, tests gated behind feature flag `real-browser-tests` and skipped by default. Ship the feature-gated tests so local runs validate.

    ```rust
    use tauri::{AppHandle, State, Channel};
    use automation::{Executor, ExecutorEvent, ChromiumoxideDriver, PlaywrightSidecarDriver};

    #[tauri::command]
    pub async fn launch_automation(
        app: AppHandle,
        state: State<'_, AppState>,
        story_source: String,
        project_folder: String,
        on_event: Channel<ExecutorEvent>,
    ) -> Result<(), AppError> {
        let parse = story_parser::parse(&story_source);
        let story = parse.ast.ok_or_else(|| AppError::InvalidArgument("no story".into()))?;
        // Launch chromiumoxide directly (in-process)
        let chromium = ChromiumoxideDriver::launch(LaunchConfig::from_meta(&story.meta)).await?;
        // Launch playwright sidecar via tauri-plugin-shell
        let sidecar_cmd = app.shell().sidecar("playwright-sidecar")?;
        let (sidecar_rx, mut child) = sidecar_cmd.spawn()?;
        let playwright = PlaywrightSidecarDriver::from_child(child, sidecar_rx).await?;
        // Open project db for persistence
        let project_db = storage::ProjectDb::open(&std::path::PathBuf::from(&project_folder))?;
        let mut events = Executor::run(story, Box::new(chromium), Box::new(playwright), Some(Arc::new(Mutex::new(project_db))));
        while let Some(evt) = events.recv().await {
            on_event.send(evt).map_err(|e| AppError::Internal(e.to_string()))?;
        }
        Ok(())
    }
    ```
