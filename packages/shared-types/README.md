# @storycapture/shared-types

Shared TypeScript surfaces used across StoryCapture packages and apps.

## What Lives Here

- `src/browser-presets.ts` and `browser-presets.json`: browser/device preset
  data.
- `src/web-account.ts`: desktop/web account DTOs.
- `src/ipc.ts`: checked-in host IPC compatibility surface for the
  Tauri-compatible invoke bridge.
- `src/generated/effects.ts`: generated effect graph/render types.

## Generated / Compatibility Files

`src/ipc.ts` is intentionally checked in because the renderer still consumes the
host IPC compatibility surface while Electron implements the host side.

`src/generated/effects.ts` is generated by `ts-rs` and says so in its header.
Do not hand-edit generated effect types unless you also update the generation
source/provenance and consumers.

## Commands

This package currently has no package scripts. Verify changes through affected
consumers or root typecheck:

```bash
pnpm typecheck
```
